# 🌍 live-translator - Real-Time Audio Translation on Mac

## 🧭 Overview

live-translator shows live subtitles for audio that plays on your Mac. It helps you follow YouTube videos, podcasts, calls, and meetings in real time.

It works on macOS and uses on-screen text to show translated speech as the audio plays. This makes it easier to understand content in another language without pausing or replaying audio.

## ✅ What You Can Use It For

- Watch videos with live translated subtitles
- Follow meetings in another language
- Listen to podcasts and read the text on screen
- Capture system audio from your Mac
- See speech as text while the audio plays
- Translate audio into a language you can read

## 💻 System Requirements

Before you install live-translator, make sure your Mac meets these basic needs:

- macOS 13 or later
- Apple Silicon or Intel Mac
- Microphone access if you want to use voice features
- Screen capture permission
- A stable internet connection for translation services
- Enough free disk space for the app and temporary files

For the best result, close apps that use a lot of memory before you start.

## 🛠️ First-Time Setup

When you open live-translator for the first time, macOS may ask for a few permissions.

### Screen Recording
The app needs screen recording access to capture system audio on macOS. Allow this so it can read the audio stream from your system.

### Microphone
If the app asks for microphone access, allow it. Some speech tools may use it for better recognition or input checks.

### Notifications
Turn on notifications if you want to see status updates when translation starts, stops, or changes.

### Language Choice
Set your source language and target language before you start. For example:

- Source: English
- Target: Spanish

If you are not sure, choose the language you hear most often in the audio.

## ▶️ How to Use It

1. Open live-translator
2. Select the audio source on your Mac
3. Choose the language you want to translate from
4. Choose the language you want to read
5. Start playback in YouTube, podcasts, or your meeting app
6. Watch the translated text appear on screen
7. Adjust the text size or window position if needed

If the text moves too fast, lower the audio speed or pause the source for a moment

## 🧩 Common Problems

### Nothing appears on screen
Check that:
- the app is running
- screen recording access is enabled
- the correct audio source is selected
- the audio is playing

### Text is wrong
Check that:
- the source language is correct
- the audio is clear
- the speaker has a strong accent or background noise is low

### App will not open
Check that:
- the file finished downloading
- macOS allowed the app
- the app is in your Applications folder

### Audio is delayed
Try:
- closing other heavy apps
- using a stronger internet connection
- restarting the app
- lowering the load from other background tools
